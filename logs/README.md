# Logs Directory

This directory contains logs generated by the application. Each log file contains a detailed record of events and operations performed by the system.

## Automatically Created Log Files

The log files are automatically created as the application runs and typically include:
- **application.log**: Records general application events and errors.
- **debug.log**: Contains debug information intended for development and troubleshooting.
- **access.log**: Tracks user access and actions performed within the application.

Each log file is timestamped and may rotate based on the application's logging configuration. Please refer to the application's documentation for more details.
